CSV Format Requirements
Required Columns (in order):
  1. Accessory Name
  2. manufacturer
  3. Condition
Note: Any content is acceptable as long as headers are correct. Empty fields will use defaults.
Example Accessory Names:
  • Tripod
  • Tracking Rod
  • External Radio
  • Reflector
  • Levelling Staff
  • GNSS Battery
  • Pole
  • Mini Prism
  • Any custom accessory name
Example Manufacturers:
  • Hi Target
  • Topcon
  • Leica
  • Sokkia
  • Any manufacturer name
Example Conditions:
  • Good
  • New
  • Second Hand
  • Needs Repair
  • Bad
  • Any condition description
Note: Any accessory name, manufacturer, and condition from your CSV will be accepted.
